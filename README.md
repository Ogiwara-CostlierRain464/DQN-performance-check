# About
hhlabの最終課題で使うコード。Q tableに配列を使う手法と、NNを使う手法の比較を行う。

本に記載されているコードには以下の問題があり、扱いにくいものとなっていた。
- Brainクラスに多態性がない
- observation, stateなど同じ意味の用語が別表記になって混ざっている
- 他、Type hintingが無い等

なので、それらの点を改善したコードを記述した。

例えば、Agentクラスのコンストラクタにて、Q tableの実装を`TableBrain`と`NNBrain`からすぐに選択できるようになっている。

コードのエントリポイントはapp.py。
